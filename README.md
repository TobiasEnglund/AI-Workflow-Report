# Workflow for AI projects

This repository contains a (Swedish) school report that explores the workflow of a machine learning app using linear regression. The provided scenario involves predicting house prices based on various features such as the number of rooms, location, size, etc.

## The report contains:

- **Data Collection**: Data formats, storage options, and potential sources for collecting house pricing data.
- **Data Visualization**: Methods for identifying patterns and anomalies.
- **Data Preprocessing**: The steps needed to prepare the data for training.
- **Linear Regression**: How linear regression works and how it applies to the housing prices scenario.
- **Model Deployment**: The steps involved in deploying a machine learning model.
- **Technologies Used**: The technologies that could be used in this scenario.
